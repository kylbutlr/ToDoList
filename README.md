# Todo List (API)
### Todo list application REST API
##### Features: CRUD, routing, and tests
##### Using: PostgreSQL, Express, and Jest
### [Can be used with Todo List (React)](https://github.com/kylbutlr/todo-list-react)

## Install

```bash
npm install
```

## Usage

```bash
npm start
```

Runs on port `3000`
